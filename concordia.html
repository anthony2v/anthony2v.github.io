Concordia University

In this page I will describe my experience while studying at Concordia University in Montréal, Québec. Specifically, I will be talking about computer science and software engineering classes that I took as well as the content and learning activities that I took part in. I will mention anything related to the teachers that I had, since the experience depends on when I took the course and what section I was in. A potential or current student might have a completely different teacher and therefore a different experience compared to my own.

COMP 228 - System Hardware

This course is about the fundamentals of computer organization and architecture. If you do not know what those are, do not worry because you will learn about them in this class. You will learn about circuits, the binary number system, and how it applies to computers. This course is a prerequisite for COMP 346 as that class builds on the concepts learned in this one.

COMP 232 - Mathematics for Computer Science

Discrete math is what you should be thinking about when you think about this class. You will learn about proofs, sets, number theory and mathematical induction. Almost all of the topics learned here come back in future classes, including anything algorithm-related. If you are considering taking math courses such as COMP 339 (Combinatorics), seriously pay attention in this class.

COMP 233 - Probability and Statistics for Computer Science

Another math class for future computer scientists. Topics include introduction to, well, probability and statistics, and then goes into various theorems and random variables. The end of the course deals with hypothesis testing, regression analysis and random number generation. The subject resurfaces in a few classes such as COMP 445 (Networking) and video game related courses.

COMP 248 - Object-Oriented Programming I

For some students this is probably their first ever programming class. The course is taught in Java and you will learn the basics of programming in general and a little bit about object-oriented programming. When I say basics, I mean things like variables, control flow, operators, and later on, objects, members, and more.

COMP 249 - Object-Oriented Programming II

This is the follow up class to COMP 248. Here you will learn more about objects and access rights between them. Towards the end you will learn fundamental about recursion, array lists, and linked lists.

Assignments
COMP 335 - Introduction to Theoretical Computer Science

While technically this is a math class, you will not be dealing with numbers. It starts out with simple finite automata, or state machines, and builds on that little by little. You will eventually have to work on proofs, specifically to prove that a grammar is unambiguous or not. Then you will work with automata that use stacks and finally you will learn about turing machines.

COMP 345 - Advanced Program Design with C++

This is a very good project class. It is basically COMP 248, 249 and part of 352 mixed together with some design patterns (strategy and adapter). The entire semester is a cumulative project, which is usually a text-based version of Risk, a variant of Risk, or a more recently released board game. The first few weeks contain COMP 248 and 249 but in C++. It is assumed that you have enough experience with programming that you can pick up a new language fairly easily.

COMP 346 - Operating Systems

This course is all about operating system concepts. You will learn topics such as CPU scheduling, threading, semaphores, and synchronization. Concepts from COMP 228 come back in this course, particularly those concerning CPU and memory. It is an important course because in later courses threading and synchronization will return, no matter which career path you go into.

COMP 348 - Principles of Programming Languages

If you have ever heard of the word polyglot then you should think of this course. A polyglot is someone who is an expert in multiple programming paradigms, which is exactly what you will be studying in this course. Around 4 programming languages are taught in this course depending on the teacher. When I took it, I learned Prolog (Logical), Lisp (Functional), C (Procedural), and Ruby (Multi-Paradigm). Some teachers will also teach advanced Java topics including aspect-oriented programming. Do not worry, you only scratch the surface of each language, it is not meant to be a super heavy course.

Assignments
COMP 352 - Data Structures and Algorithms

This is maybe the most important that you will take during your time at Concordia. Be sure to learn the material here very well, because it comes back in so many other courses. You will learn about abstract data types, as well as various containers used in everyday applications, including stacks, queues, sets, lists, trees, maps and graphs. Algorithms related to these structures will also be taught as well as how to calculate their performance, which is their time and space complexity. The assignments and labs involve solving programming problems using what you have learned in the lectures.

Assignments
